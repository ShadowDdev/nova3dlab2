# Nova 3D Lab — Documentation

## Website Overview
Nova 3D Lab is a premium 3D printing service and e-commerce platform built for the Indian market.

- **Brand**: Nova 3D Lab
- **Location**: Chandigarh, Chandigarh, India
- **Currency**: Indian Rupees (₹)
- **Support**: support@nova3dlab.com
- **Website**: https://nova3dlab.com (coming soon)

## Tech Stack
- React 18 + Vite + TypeScript
- Tailwind CSS + shadcn/ui
- Supabase (Auth, PostgreSQL, Storage, Realtime)
- Zustand (cart, wishlist, auth state)
- TanStack Query v5
- Framer Motion + React Three Fiber
- Stripe (INR payments)

## Key Features Already Live
- Instant 3D model upload & volume calculation
- Real-time pricing engine
- 3D model viewer (drag to rotate + AR)
- Cart with persistence
- Stripe checkout (INR)
- Dark mode
- Mobile-first design

## Environment Variables (.env.example)
```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_anon_key
VITE_STRIPE_PUBLISHABLE_KEY=pk_live_...
VITE_SITE_URL=https://nova3dlab.com
```
